## 🛠️ what i'm building

- **Stocky** 📈 a Discord economy bot with its own currency and a full stock market (order book + AMM trading)
- **Splitty** 🧩 file encryption/obfuscation tool that scatters encrypted chunks across your filesystem like a digital horcrux
- **Vesper** ⏳ time-locked encryption app using a randomness beacon, basically "open this file in the future, not before"
- **osu!vis (soundvis)** 🎯 a custom osu! ruleset where notes fly inward to a center hit circle, made this bc i wanted rhythm games to feel more chaotic in a good way
